# hw4 - socketio
Implement Socket IO with React/vite to to achieve a basic chat application

## Setting Up
1. Download the zip file and unzip
2. Install modules
```
cd hw4_socketio
npm install
cd frontend
npm install
```
3. To start running the web server:
```
cd ..
npm start
```
4. Visit http://localhost:3000/
